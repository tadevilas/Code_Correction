# Code Correction

An early-stage scaffold for a code correction / review application.

## Repository Layout

- **app.py** – Application entry point (in progress)
- **templates.py** – Prompt templates used for code correction logic
- **src/** – Core package (`helper.py` for utility functions)
- **research/** – Exploratory notebooks and research material
- **requirements.txt** – Project dependencies
- **setup.py** – Package setup

## Status

This project is in an early scaffolding stage — most files are placeholders to be filled in as the code correction logic is built out.

## Setup

```bash
pip install -r requirements.txt
```
